# merge_train demo target

This directory is a test playground for the
[merge_train](https://github.com/jleechanorg/merge_train) symbol-level
lock registry.

`multi_func.py` declares four disjoint top-level functions — `alpha`,
`beta`, `gamma`, `delta`. Three demo workers each reserve a disjoint
symbol and open a PR that edits only their own function. A fourth
worker tries to overlap on `alpha` and is denied by the lock log.

The corresponding `file_domains.yaml` maps files to domains for symbol-level locks. Specifically:
- `merge_train_demo/multi_func.py` is mapped to the `demo` domain, owned by `@jleechan2015`.
- `merge_train_demo/multi_func_2.py` is mapped to the `demo2` domain, owned by `@jleechan2015`.

Example entries in `file_domains.yaml`:
```yaml
domains:
  demo:
    paths:
      - "merge_train_demo/multi_func.py"
    owners:
      - "@jleechan2015"

  demo2:
    paths:
      - "merge_train_demo/multi_func_2.py"
    owners:
      - "@jleechan2015"
```
